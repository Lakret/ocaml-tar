Module Tar.Make.Archive

module Archive: sig .. end

Utility functions for operating over whole tar archives
val with_next_file : Tar.IO.in_channel -> (Tar.IO.in_channel -> Tar.Make.Header.t -> 'a) -> 'a
Read the next header, apply the function 'f' to the fd and the header. The function should leave the fd positioned immediately after the datablock. Finally the function skips past the zero padding to the next header
val list : ?level:Tar.Make.Header.compatibility ->
       Tar.IO.in_channel -> Tar.Make.Header.t list
List the contents of a tar
val extract_gen : (Tar.Make.Header.t -> Tar.IO.out_channel) -> Tar.IO.in_channel -> unit
extract_gen dest extract the contents of a tar. Apply 'dest' on each header to get a handle to the file to write to
val create_gen : ?level:Tar.Make.Header.compatibility ->
       (Tar.Make.Header.t * (Tar.IO.out_channel -> unit)) Stream.t ->
       Tar.IO.out_channel -> unit
Create a tar on file descriptor fd from the stream of headers.
val copy_n : Tar.IO.in_channel -> Tar.IO.out_channel -> int64 -> unit
This function is DEPRECATED. copy_n ifd odf n copies exactly n bytes from ifd to ofd
val skip : Tar.IO.in_channel -> int -> unit
This function is DEPRECATED. skip fd n reads and throws away n bytes from fd
